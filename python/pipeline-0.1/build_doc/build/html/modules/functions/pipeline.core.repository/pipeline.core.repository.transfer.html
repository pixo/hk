pipeline.core.repository.transfer¶

pipeline.core.repository.transfer(sources=[], destination='', doc_id='', rename=True)[source]¶

This function create the asset user path of a particular asset. :param doc_id: The asset code. :type doc_id: str :returns: str/bool – Return the workspace path if path is created else False .

Example:

>>> createWorkspace ( doc_id = "prod_chr_mickey_mod_a" )
>>> '/homeworks/users/jdoe/projects/prod/chr/mickey/mod/a'
