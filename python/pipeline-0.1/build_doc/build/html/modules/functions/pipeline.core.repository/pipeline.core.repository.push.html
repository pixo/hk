pipeline.core.repository.push¶

pipeline.core.repository.push(db='', doc_id='', src_ls=[], description='', progressbar=False, msgbar=False, rename=True)[source]¶

This function copy the desired file from local workspace to repository.

Parameters:
  • db (Database) – the database
  • doc_id (str) – The asset code
  • src_ls (str/list of str) – The list of files to push
  • description (str) – The file extension
  • progressbar (PySide messagebar) – The pyside progress bar
  • msg – The pyside message bar
  • rename (bool – if True rename the file(s)) – Rename the file (default True)
Returns:

str – Return the published directory

Example:

>>> db = utils.getDb()
>>> push ( db = db, doc_id = "bls_chr_belanus_mod_main", "/homeworks/users/jdoe/projects/bls/chr/belanus/mod/main/file_to_push.mb" )
