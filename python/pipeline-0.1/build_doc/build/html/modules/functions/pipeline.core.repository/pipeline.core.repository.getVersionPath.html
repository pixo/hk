pipeline.core.repository.getVersionPath¶

pipeline.core.repository.getVersionPath(db=None, doc_id='', version='last')[source]¶

This function return the asset path of a particular version.

Parameters:
  • db (Database) – the database
  • doc_id (str) – The asset code.
  • version (str/float/int – ‘last’ or 1,2,3,4 etc) – The asset code.
Returns:

str – the asset directory

Example:

>>> db = utils.getDb()
>>> getVersionPath ( db = db, doc_id = "bls_chr_belanus_mod_main", version = "last" )
>>> '/homeworks/projects/bls/chr/belanus/mod/main/008'
