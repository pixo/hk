pipeline.core.asset.createTask¶

pipeline.core.asset.createTask(db=None, doc_id='', description='', overdoc={})[source]¶

This function create a task into the provided database.

Parameters:
  • db (couchdb.client.Database) – The database.
  • doc_id (str) – The asset code.
  • description (str) – The asset description.
  • overdoc (dict) – A dictionnary that contains extra document attributes.
Returns:

document – The database document.
Raises :

AttributeError, KeyError

Example:

>>> db = pipeline.utils.getDb ( dbname = "prod" , serveradress = "127.0.0.1:5984" )
>>> createTask ( db = db, doc_id = "prod_ch_mickey_mod_a", "This is the mickey modeling task 'a'" )
