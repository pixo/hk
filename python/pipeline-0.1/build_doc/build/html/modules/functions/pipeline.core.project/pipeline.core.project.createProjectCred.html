pipeline.core.project.createProjectCred¶

pipeline.core.project.createProjectCred(name, db_server, host_root)[source]¶

This function create a project credential file. It contains the project database server adress and the host root adress.

Parameters:
  • name (str) – The project name.
  • db_server (str) – The data base server adress.
  • host_root (str) – The host server adress root path.
Returns:

str/bool – If credential file created return the file path else False

Example:

>>> createProjectCred ( name = 'prod', db_server = 'admin:pass@192.168.0.100:5984', host_root = 'admin@192.168.0.9:/homeworks' )
>>> '/homeworks/users/jdoe/.hk/prod'
