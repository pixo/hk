pipeline.core.project.createProject¶

pipeline.core.project.createProject(name='', description='Default', db_server='', host_root='', overdoc={})[source]¶

This function create a project.

Parameters:
  • name (str) – The project name
  • description (str) – The project description
  • db_server (str) – The data base adress
  • host_root (str) – The host data server root adress
  • overdoc (dict) – A dictionnary that contains extra document attributes.
Returns:

couchdb.client.Database – return the db.
Raises :

AttributeError, KeyError

Example:

>>> createProject ( name = "prod", description = "this is the project prod",
                    db_server = "admin:pass@127.0.0.1:5984", host_root = "admin@127.0.0.1:/homeworks" )
