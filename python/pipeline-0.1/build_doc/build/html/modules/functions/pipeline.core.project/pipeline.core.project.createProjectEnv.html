pipeline.core.project.createProjectEnv¶

pipeline.core.project.createProjectEnv(name='')[source]¶

This function create a project environment file. It contains project environment variables related to the project. This file is sourced each times a user log to a project via the hk-project command.

Parameters:name (str) – The project name.
Returns:str/bool – If environment file created return the file path else False

Example:

>>> createProjectEnv ( name = "prod" )
>>> '/homeworks/projects/prod/config/prod.env'
