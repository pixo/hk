pipeline.core.project.getProjectUsers¶

pipeline.core.project.getProjectUsers(db=None)[source]¶

This function return the list of authorised project users.

Parameters:db (couchdb.client.Database) – The database.
Returns:list – Return the list of authorised project users.

Example:

>>> db = pipeline.utils.getDb ( dbname = "prod" , serveradress = "127.0.0.1:5984" )
>>> getProjectUsers ( db )
>>> [ 'jdoe', 'mickey', 'pixo' ]
