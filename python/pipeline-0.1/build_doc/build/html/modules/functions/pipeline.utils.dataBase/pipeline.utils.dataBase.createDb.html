pipeline.utils.dataBase.createDb¶

pipeline.utils.dataBase.createDb(name=None, serveradress=None)[source]¶

This function create a DataBase into the provided server.

Parameters:
  • name (str) – The database name.
  • serveradress (str) – The asset code.
Returns:

Database – return the database
Raises :

DatabaseError if database already exist

Example:

>>> createDb ( name = "prod", serveradress = "admin:pass@192.168.0.100" )
