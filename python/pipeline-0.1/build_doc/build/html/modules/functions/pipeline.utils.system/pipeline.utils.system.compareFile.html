pipeline.utils.system.compareFile¶

pipeline.utils.system.compareFile(file_a='', file_b='')[source]¶

This function compare two files contains based on sha1. It is very useful if you need to know if two files are the same.

Parameters:
  • file_a (str) – The first file path
  • file_b (str) – The second file path
Returns:

bool – True if file are the same.
Raises :

RepositoryError if one of the file doesn’t exists.

Example:

>>> compareFile ( file_a = "/home/user/filea", file_b = "/home/user/fileb" )
>>> True
